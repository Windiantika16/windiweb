HTML

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web dan menampilkan berbagai informasi di dalam browser. HTML menggunakan tag-tag untuk memberikan instruksi kepada browser mengenai bagaimana halaman web seharusnya ditampilkan.

Sejarah HTML

HTML pertama kali diciptakan oleh seorang fisikawan di CERN, Swiss bernama Tim Berners-Lee pada tahun 1991. Tujuannya adalah untuk mempermudah pembagian informasi melalui web browser.

Beberapa tahun penting dalam sejarah HTML adalah:

  • HTML Versi Pertama dirilis pada tahun 1993, versi pertama HTML berisi 18 tag.
  • HTML 2.0 atau RFC1866 disetujui sebagai standard pada 22 september 1995.
  • HTML 3.2 dirilis pada 14 januari 1996.
  • HTML 4.0 dirilis pada 18 Desember 1997, versi ini memiliki perintah dasar untuk menambahkan gambar, tabel, tautan, teks, dan formulir.
  • ISO/IEC 15445:2000 "ISO HTML" berdasarkan pada HTML 4.01 strict- 15 mei 2000.
  • XHTML 1.0 (2000)
  • XHTML 1.1 diliris pada 16 februari 2007.
  • HTML 5 dirilis pada 2014, versi ini merupakan perpanjangan dari HTML 4.01 dan bisa menggabungkan HTML dengan bahasa pemrograman lainnya seperti CSS dan JavaScript.

Ciri-ciri Dokumen HTML

  • Ekstensi file berupa .html atau .htm
  • Non case sensitive
  • Terdiri dari tag pembuka dan penutup
  • Tag saling berpasangan dan bersarang

Struktur HTML


        <html>
            <head>
                <title>Judul Halaman</title>
            </head>
            <body>
                <p>Ini adalah konten halaman.</p>
            </body>
        </html>

Tag HTML

Tag, atribut, dan elemen adalah bagian penting dari HTML:

  • Tag pembuka: <tag>
  • Tag penutup: </tag>
  • Single tag: <br />, <hr />

Font Tag

Tag ini digunakan untuk memodifikasi teks:

  • <b>Teks tebal</b>
  • <i>Teks miring</i>
  • <u>Teks bergaris bawah</u>
  • <sup>Superscript</sup>
  • <sub>Subscript</sub>

Image Tag

Untuk menampilkan gambar pada halaman web:

<img src="gambar.jpg" alt="Deskripsi gambar" width="300" height="200" />
