HTML

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web dan menampilkan informasi di browser.

Sejarah HTML

HTML pertama kali diperkenalkan oleh Tim Berners-Lee pada tahun 1991.

  • 1993: Versi pertama HTML dirilis.
  • 1995: HTML 2.0 diresmikan sebagai standar.
  • 1996: HTML 3.2 muncul.
  • 1997: HTML 4.0 hadir.
  • 2014: HTML5 dirilis.

Ciri-ciri Dokumen HTML

  • Ekstensi file .html atau .htm
  • Tidak case-sensitive
  • Berpasangan antara tag pembuka dan penutup
  • Tag bisa bersarang satu sama lain

Struktur HTML

<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        <p>Ini adalah konten halaman.</p>
    </body>
</html>

Tag HTML

  • <h1> hingga <h6>: Tag heading
  • <p>: Tag paragraf
  • <br>: Tag untuk baris baru (single tag)

Font Tag

  • <b>: Teks tebal
  • <i>: Teks miring
  • <u>: Teks bergaris bawah

Image Tag

Tag ini digunakan untuk menampilkan gambar:

<img src="gambar.jpg" alt="Deskripsi" width="300" />

Anchor / Hyperlink Tag

<a href="http://website.com">Kunjungi Website</a>
